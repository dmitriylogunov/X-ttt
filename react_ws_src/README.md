- react

- babel
- react-router
- ampersand
- sass

### Getting Started

Install dependencies (Node 22+ is required):

~~~bash
npm install
~~~

Start the Vite dev server at [http://localhost:3000](http://localhost:3000):

~~~bash
npm run dev
~~~

Lint the source:

~~~bash
npm run lint
~~~

Execute the Jest suite:

~~~bash
npm run test
~~~

Generate a production build (outputs to `dist/`):

~~~bash
npm run build
~~~
